# Uninstall global hotwater script

Uninstall global hotwater script

## Usage

``` r
uninstall_hotwater(install_folder)
```

## Arguments

- install_folder:

  folder to uninstall hotwater from.

## See also

[install_hotwater](https://elianhugh.github.io/hotwater/reference/install_hotwater.md)

## Examples

``` r
if (FALSE) { # interactive()
 hotwater::uninstall_hotwater()
}
```
