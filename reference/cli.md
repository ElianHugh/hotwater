# Run hotwater from the command line

Following
[`install_hotwater()`](https://elianhugh.github.io/hotwater/reference/install_hotwater.md),
the `hotwater` command can be used to run a hotwater engine straight
from the terminal. See
[`run()`](https://elianhugh.github.io/hotwater/reference/run.md) for
further details on default values.

`hotwater -v` will provide the current version of hotwater.
`hotwater -h` will provide help text.

## Arguments

- -f:

  plumber file

- -d:

  extra directories

- -p:

  plumber port
